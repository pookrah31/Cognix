# Cognix Cognitive Suite

**Cognix** is a premium, web-based cognitive training suite designed to challenge and sharpen your memory, processing speed, focus, and typing accuracy. Built using pure HTML5, vanilla CSS3, and Javascript.

---

## 🎮 Game Modules

### 1. 🧠 Reverse Recall (Working Memory)
- Memorize numeric sequences as they light up, then input them in reverse order.
- Higher levels introduce faster sequences and floating visual distractors to test your attention filtering.

### 2. 🎨 Color Clash (Cognitive Flexibility)
- Based on the famous psychological **Stroop Effect**.
- Speed-challenge: choose the color of the ink the word is printed in, overriding your automatic response to read the word text.

### 3. 🔲 Pattern Matrix (Spatial Memory)
- A grid matrix flashes a sequence of coordinates.
- Memorize the positions and tap the correct cells from memory. Grid sizes scale dynamically up to 5x5.

### 4. ➕ Math Blitz (Arithmetic Speed)
- Rapid-fire arithmetic operations (addition, subtraction, multiplication) under a timer.
- Scale your score by answering as fast as possible to secure speed-bonus points.

### 5. ⌨️ Type Rush (Motor Speed & Accuracy)
- A typing speed tutor that challenges you to type words, focus quotes, and neuro-scientific statements.
- Tracks real-time **WPM** (Words Per Minute) and character **Accuracy %**.

---

## ✨ Features

- **Premium Glassmorphic Design**: Clean slate-navy backdrop, translucent slate cards, and soft ambient watercolor meshes.
- **🏆 Personal Bests Modal**: View your highest records across all 5 training modules in a single clean pop-up window.
- **⚡ Dynamic Profile Badge**: Unlock badge ranks (from *Novice* to *Grandmaster Cognition*) based on your combined high scores.
- **🎯 Daily Objectives**: Clear milestones (e.g. "Reach Lvl 4 in Pattern Matrix") that update progress indicators automatically.
- **🔊 Synthesized Audio Effects**: Built-in sound signals using Web Audio API (no external asset downloads required!).
